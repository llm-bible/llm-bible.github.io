# Awesome Large Language Models!

### A living literature review for the Large Language Model field

This is the repository for the living literature review of the field of Large Language Models [https://llm-papers.github.io/](https://llm-papers.github.io/). LLMs have taken the world by storm, providing a step chance in performance of AI understand of natural language, images and audio and other modalities of data. Please 
look at the website for more information and how to contribute your work or taxonomy to the website. 

### Contributing

This research area is evolving so fast that a static review cannot keep up. But a website can! We hope to make this site a living document. Anyone can add a paper to this web site, essentially by submitting a web form with their content.
